### Commands
| Number | Description             |
|--------|-------------------------|
| 0      | Set LEDs                |
| 1      | LED counts              |
| 2      | Write channel           |
| 4      | Blink color             |
| 5      | Reset time              |
| 10     | Start sequence          |
| 11     | Pause sequence          |
| 12     | Stop sequence           |
| 13     | Sequence EEPROM address |
| 14     | New sequence            |
| 15     | Add sequence step       |

Basic commands: [Module readme](src/hc-module/README.md)
